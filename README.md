# Chat AI

An App like Gemini

## tech
- framework: nextjs
- ui: shadcn
- deployment: cloudflare
- database: d1
- authentication: firebase
- storage: r2
- payment-gateway: xendit
- orm: drizzle
- testing: playwright

## feature
- login with google
- list history chat
- chat with ai
- user can pay for premium (free: 50mb storage, 50ask/day, premium: 100mb, 200ask/day)
